# Fish Weight Prediction

## Project Overview
This project predicts fish weight using machine learning based on physical measurements such as length, height, and width.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation

## Results
- Model: Linear Regression
- Evaluation Metric: R² Score
